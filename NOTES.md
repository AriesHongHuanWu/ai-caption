# Local Studio v0.1.38

⬇️🧰 **Cleaner Downloader + Toolbox.**

### Downloader — own place, clearer layout
- The **Downloader** now lives in its **own sidebar category** ("Get media"), separate from Audio and Video.
- The screen is reorganised into clear steps — **① paste a URL → ② choose audio *or* video (segmented toggle) + format → ③ done: file actions + what's next** — so it's not a wall of options. Every feature is still there.
- **Audio and video** are both first-class: a clear Audio/Video switch, format options with a one-line hint, and download history is now a tidy collapsible list.

### Toolbox — file first
- When you pick a tool, the file drop and that tool's controls sit at the **top**, with the tool-switch buttons below — no scrolling to run a tool.

Also: the in-app update dialog now shows the **real release notes** (it was reading a hardcoded placeholder).

Everything from v0.1.37 is included: mastering that actually improves the track + 21 genre/hip-hop/artist-style presets (savable), and the song analyzer (key / BPM / structure / EQ / vocal-mix advice).

> The Downloader + song analysis need the local engine to include `yt-dlp`. If the Download mode shows an "engine missing" notice, run Settings → repair/reinstall engine once. Mastering and the Toolbox work immediately.
