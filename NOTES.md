# Local Studio v0.2.1

## Editor moved to Beta (opt-in)

The in-app video editor (剪輯室) is being reworked, so it now ships **disabled by default**. To try the work-in-progress editor, open 剪輯室 and **enter the code “beta”** — it then unlocks and is remembered on your machine.

This lets the editor keep improving without getting in the way of the audio/mastering, lyrics, subtitle and visualizer tools, which are all unchanged.
